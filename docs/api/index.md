# 通用接口  
框架中自带的一些接口













### 健康检查
----
**功能：**响应健康检查请求  
**请求方式：**`GET`  
**请求地址：**  
```
/health_check
```

**返回值：**  
```
ok
```












### 错误码字典 
----
**功能：**获取错误码字典  
**请求方式：**`GET`  
**请求地址：**  
```
/error_code_maps  
```
**返回值：**  
```json
{
    "code": 0,
    "msg": "",
    "data": [], //错误码数组
}
```
